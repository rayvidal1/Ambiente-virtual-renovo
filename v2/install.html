Android

Passo a passo

  1. Abra esta pagina no Chrome.
  2. Toque em "Instalar app" se o botao aparecer.
  3. Se nao aparecer, abra o menu do Chrome.
  4. Escolha "Instalar app" ou "Adicionar a tela inicial".

Limpeza

Se a v1 travou

  1. Desinstale a versao antiga da tela inicial.
  2. Limpe os dados do site no Chrome.
  3. Abra a v2 no navegador.
  4. Instale novamente usando esta pagina.
